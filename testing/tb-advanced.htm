Sub-Goal 1: Move Navigation to the Bottom

Result: This worked like a charm! No playing with the result. It was clean!

There is a way to leverage the "always up top" aspect of navigation, so that it will stay on the top of the screen even if you scroll. Not only is this nifty, but it also has a secondary benefit of making plain HTML pages look better.

CSS styled pages turn ULs into navigation for SEO purposes, as opposed to the old fashion of using a table or images with JavaScript to load up your code up top. This also made pages load slower back in the day. But there is no real reason for SEO nor for coding that the ugly UL has to be at the top, except if the code has to live where it is placed on the page. But with jQuery, you don't need to!

The end result of this structured approach looks a bit cleaner on plain pages (like this one) and it even makes your pages important aspects load a teeny bit quicker too!
